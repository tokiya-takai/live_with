## 概要

靴好きのための管理アプリです。<br />
定期メンテナンスの必要な革靴を傷めないよう、管理をサポートします。<br />

## 開発環境

MacOS Catalina 10.15.7<br />
PHP 7.3.11<br />
Laravel Framework 8.37.0<br />
Vue.js 2.6.12<br />
Docker 20.10.5<br />
Docker Compose 1.29.0<br />
npm 7.10.0<br />
node 14.16.1<br />

## 機能

Authを使用したユーザー管理機能<br />
・新規登録・ログイン・ログアウト<br />
<br />
リストの投稿機能<br />
・新規投稿<br />
・編集機能(実装予定)<br />
・投稿削除機能(実装予定)<br />
・Vueを使用した並び替え機能(実装予定)<br />
<br />
レスポンシブデザイン<br />
・現時点では一部実装　全てにおいて実装予定<br />
<br />
その他<br />
・ロゴ　Illustratorで作成<br />

## 実装予定機能

・他ユーザーのリストの閲覧機能<br />
・公開非公開機能<br />
